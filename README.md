## UPOINT-CLI

### commands
* create
* release
* rundev
* nextver

#### create

Create a project direcotry. Run console command inside new dir:

`upoint create`

follow steps

#### release

`upoint release`

`upoint release --email=mail@mail.com --password=123456`

#### rundev

run in developer mode (hot reload supported)


#### nextver
`nextver --major`

`nextver --minor`

`nextver --build`

`nextver --revision`

`nextver --major --alpha`

`nextver --minor --beta`

`nextver --build --rc`
